1. Project Overview

Lookit ek full-stack Educational Marketplace Platform che jo students ane professional instructors/tutors ne connect kare che.

Aa ek two-sided marketplace che jya:

  • Professionals apnu educational content (books, videos, courses) create ane sell kari shake che
  • Students professionals ne discover kari, content purchase kari, ane sikhi shake che
  • Admin platform moderate kare che — approvals, payments, reviews badhu manage kare

Core Purpose: Professionals ne apni expertise monetize karva enable karvanu, students ne quality education provide karvanu, ane admin ne platform manage karva devano.

7. Key Features — Kevi rite kaam kare che

🔐 Authentication Flow

1. User signup form fill kare (name, email, password, role)
2. Password bcryptjs thi hash thay (10 rounds)
3. User PostgreSQL ma store thay
4. Professional hoy to → Admin ne approval notification jaay
5. Login → NextAuth JWT token generate thay (30 days valid)
6. Google login → Auto student account create thay

✅ Professional Approval Flow

1. Professional register kare → status: "pending"
2. Admin dashboard ma pending list ma dikhe
3. Admin approve ya reject kare (note saathe)
4. Professional ne notification aave
5. Approved hoy to j login thay ane dashboard access male
6. Rejected hoy to login block thay

📦 Content Upload Flow

1. Professional → Dashboard → Add section open kare
2. Book / Video / Class add kare
3. File Vercel Blob ma upload thay (ya YouTube URL)
4. Content library ma save thay
5. Followers ne notification jaay
6. Students ne shop / courses ma dikhe

💳 Purchase Flow

1. Student content browse kare
2. Cart ma add kare
3. Checkout → Payment details enter kare (card info)
4. Transaction ID generate thay (TXN-timestamp-random)
5. Payment record store thay
6. Student ne content access male
7. Student + Professional dono ne notification jaay

🔔 Notification System

1. Events trigger thay: purchase, follow, approval, new content
2. Respective store ma save thay (student/professional/admin)
3. Dashboard open karo to API thi fetch thay
4. Notification drawer ma dikhe
5. Read karo to "read: true" mark thay

🗺️ Map Feature

1. Professionals apnu location set kare (profile ma)
2. /professionals page par map view available che
3. Leaflet + React Leaflet thi interactive map render thay
4. Students nearby professionals find kari shake

⭐ Profile Boost System

1. Professional upgrade plan purchase kare (Razorpay thi)
2. profile_boosted_until date set thay
3. Boost active hoy tyare professional search results ma upar dikhe
4. Boost expire thay to normal position par aave
